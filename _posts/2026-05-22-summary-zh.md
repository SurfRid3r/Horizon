## [礼来 Retatrutide 三期试验减重达 28.3%](https://www.prnewswire.com/news-releases/lillys-triple-agonist-retatrutide-delivered-powerful-weight-loss-in-pivotal-phase-3-obesity-trial-302778859.html) {: #"item-1" } ⭐️ 9.0/10

礼来公布了 TRIUMPH-1 三期试验结果，显示 retatrutide 在最高 12 mg 剂量下 80 周内平均减重 28.3%。该试验在肥胖或超重成人中达到了所有剂量组的主要和关键次要终点。 这代表了肥胖治疗的重大突破，其疗效可能超过目前的 semaglutide 和 tirzepatide 等市场领导者。如此高比例的患者实现超过 30% 的减重，可能会显著改变代谢疾病的治疗范式。 该研究涉及约 2500 名患者，其中 4 mg 剂量组显示出 19.0% 的减重效果，且因不良事件停药率低于安慰剂组。副作用主要为胃肠道反应，未观察到心脏或肝脏安全问题。

telegram · zaihuapd · May 22, 02:18

**背景**: Retatrutide 是一种三重受体激动剂，同时靶向 GLP-1、GIP 和胰高血糖素受体以调节新陈代谢和食欲。虽然像 semaglutide 这样的 GLP-1 激动剂已广泛用于减肥，但添加胰高血糖素受体激活被认为能增强能量消耗。这项三期试验是获得监管批准用于广泛临床应用的关键一步。

## [FTC 就欺骗性“主动监听”AI 营销指控达成和解](https://simonwillison.net/2026/May/22/ftc-active-listening/#atom-everything) {: #"item-2" } ⭐️ 8.0/10

FTC 与 Cox Media Group 及其他两家公司达成和解，要求其支付近 100 万美元罚款，原因是它们虚假声称其“主动监听”服务使用智能设备监听消费者对话。该机构证实，该服务实际上并未监听对话或使用语音数据，而是转售来自数据经纪人的电子邮件列表。 此项和解确立了对欺骗性监控营销行为的重要监管先例，并明确指出将同意条款隐藏在服务条款中对于侵入性数据收集是非法的。这也正式证实了备受争议的“主动监听”广告技术（如其营销所言）实际上并未投入使用，有助于消除消费者的普遍担忧。 FTC 强调，点击强制性的服务条款并不构成从消费者家中收集语音数据的“选择加入同意”。此外，这些公司未能准确地将广告投放到客户期望的位置，这进一步增加了其服务的欺骗性。

rss · Simon Willison · May 22, 04:48

**背景**: “主动监听”指的是一种有争议的营销概念，公司声称通过分析来自智能设备的实时语音数据，根据口头对话来定向投放广告。这一概念利用了消费者长期以来对手机和智能扬声器窃听其隐私的担忧，这种担忧通常由巧合的广告定向所引发。Google、Meta 和 Amazon 等主要科技巨头一直否认使用这种侵入性技术进行广告投放。

## [字节跳动开源 3B 统一多模态模型 Lance](https://mp.weixin.qq.com/s/Xbfq72cr1796RZxJIs3L1A) {: #"item-3" } ⭐️ 8.0/10

字节跳动发布了 Lance，这是一个轻量级的 3B 参数统一多模态模型，采用 Apache 2.0 许可证。它在一个架构中原生集成了图像和视频的理解、生成以及编辑能力。 这一发布通过提供一个紧凑的开源模型，让高性能多模态 AI 变得更加普及，该模型可与大型专有系统相媲美。它使开发者能够构建涉及视觉理解和创作的复杂应用，而无需管理多个独立的模型。 Lance 采用共享上下文和双流专家架构，使用 Qwen2.5-VL 处理理解任务，使用 Wan2.2 处理生成任务。它在 GenEval 和 VBench 等基准测试中取得了最先进的结果，并利用模态感知位置编码来处理序列边界。

telegram · zaihuapd · May 22, 06:40

**背景**: 多模态 AI 模型通常需要独立的系统来进行理解和生成，这增加了复杂性和资源消耗。双流专家架构是指在融合之前，在不同的流中处理不同类型的数据，例如视觉和文本信息。此处使用的组件 Wan2.2 是阿里巴巴开发的开源视频生成模型，它采用了混合专家架构。

## [Cloudflare 故障报告：全球网络故障 25 分钟，影响 28% 流量](https://t.me/zaihuapd/41527) {: #"item-4" } ⭐️ 8.0/10

Cloudflare 发布了一份关于全球网络故障的报告，该故障持续了 25 分钟，影响了 28% 的 HTTP 流量。此次事故是由于在修复 Next.js 安全漏洞时，WAF 配置错误引起的。 此次事件凸显了部署紧急安全补丁所带来的风险，以及关键基础设施中配置错误可能引发的连锁反应。这对 DevOps 和安全团队来说是一个至关重要的教训，揭示了全球网络的脆弱性。 此次故障特别影响了使用旧版 FL1 代理并部署了 Cloudflare 托管规则集的客户。根本原因是在处理与 React Server Components 相关的 CVE-2025-55182 漏洞时触发的。

telegram · zaihuapd · May 22, 16:15

**背景**: Cloudflare 的 Web 应用防火墙 (WAF) 通过过滤恶意流量来保护网站，通常使用由 Cloudflare 预配置的托管规则集。React Server Components (RSC) 允许开发者在服务器上渲染 UI 组件，以提高性能并减少客户端 JavaScript。FL1 代理指的是 Cloudflare 较旧一代的代理架构。
